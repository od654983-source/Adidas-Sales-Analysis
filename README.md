#  Adidas Sales Analysis using SQL

##  Project Overview
This project analyzes Adidas sales data using SQL to extract meaningful business insights related to sales performance, profitability, products, regions, and sales methods.

The goal of this project is to demonstrate practical SQL skills by answering real-world business questions using a single sales dataset.

---

##  Dataset Description
The dataset contains transactional sales data with the following columns:

- Retailer  
- Retailer ID  
- Invoice Date  
- Region  
- State  
- City  
- Product  
- Price per Unit  
- Units Sold  
- Total Sales  
- Operating Profit  
- Operating Margin  
- Sales Method  

---

##  Tools & Technologies
- SQL Server  
- SQL (SELECT, WHERE, GROUP BY, ORDER BY)  
- Aggregate Functions  
- Date Functions  
- Window Functions (RANK)  

---

##  Business Questions & SQL Analysis

###  Overall Sales Performance
- Total Sales
- Total Units Sold
- Number of Sales Transactions

###  Retailer & Product Analysis
- Total sales by retailer
- Top 5 products by total sales
- Products with above-average operating margin

###  Regional Performance
- Total profit by region
- Top-performing state by sales
- Most profitable city

###  Time-Based Analysis
- Monthly sales trends
- Yearly sales comparison

###  Sales Method Evaluation
- Best sales method by total sales
- Most profitable sales method

###  Advanced Analysis
- Top-selling product in each region using window functions
- Ranking states based on total sales
